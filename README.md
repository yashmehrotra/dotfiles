# dotfiles

Pre Install:

1. Oh-my-zsh
2. Neovim
3. Terminator
4. Git
5. FZF
6. Vim-plug

Software:
1. Firefox
2. VLC
3. Dbox
4. Docker
5. Spotify

Usage:

1. Install GNU Stow
2. `stow $package`

Vim:

1. Install `vim-plug` from https://github.com/junegunn/vim-plug
2. FZF Install, pip install jedi, fzf in zshrc for rev search
2. Open `nvim` and run `:PlugInstall`
